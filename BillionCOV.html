This dataset page is for peer review only. For public, the dataset is freely available as an Open Access item from IEEE DataPort.



Note for reviewers: BillionCOV has been split into 29 parts (BillionCOV_01.txt, BillionCOV_02.txt, ..., BillionCOV_29.txt), with each CSV containing 50 million tweets, except for the last split. For ease in the review process, each of these splits has been further divided into equal 4 splits (e.g., BillionCOV_01.txt -> BillionCOV_01_part0.txt, BillionCOV_01_part1.txt, BillionCOV_01_part2.txt, BillionCOV_01_part3.txt). Further splits were done, honouring the request from the journal. Each CSV contains 6 columns. The last column is the country column, which seems to be absent but isn't. If a tweet is geotagged, then only the field contains valid country code information, else NULL. This is the reason the last column might seem absent. Note that only 0.87% of the tweet in BillionCOV are geotagged.
